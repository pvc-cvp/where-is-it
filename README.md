# 在哪呢？ (Where is it?)

> **一句话产品介绍**：一款基于本地存储的极简物品追踪与游戏化整理工具，专治“随手放、随时忘”。
> **链接**：https://pvc-cvp.github.io/where-is-it/

## 🚨 解决了什么问题？

在日常生活中，我们经常会把护照、备用钥匙、充电线等不常用但关键的物品随手一放，等到急用时却翻箱倒柜也找不到。传统的备忘录不够直观，而专业的库存管理软件又过于臃肿（Overengineering）。

“在哪呢？”通过最扁平化的数据结构（物品-位置映射），结合直观的视觉反馈，帮助用户以最低的认知成本记录和追踪物品的流转轨迹。

## ✨ 功能亮点

1. **极速录入与图文绑定**
   - 摒弃繁琐的表单，仅需“物品名称”和“存放位置”即可完成记录。
   - 支持调用摄像头拍照或上传图片，提供最直观的视觉记忆辅助。

2. **智能标签与多维视图**
   - **自动分类**：内置轻量级正则引擎，输入“感冒药”自动打上“医疗健康”标签，输入“充电线”自动归类“数码电子”，零延迟、零网络请求。
   - **多维聚合**：支持在首页按“全部物品”、“按分类”、“按地点”一键切换视图，再多物品也井井有条。

3. **位置流转时间轴 (Location History)**
   - 物品换地方了？直接更新位置。系统会自动保留历史轨迹，生成清晰的时间轴（Timeline），让你知道它曾经去过哪里。

4. **游戏化“每日一问” (Gamification)**
   - 引入核心 IP（一只情绪化的猫咪）。
   - 每次打开应用，猫咪会随机抽取一件物品对你进行“突击检查”。
   - 答对可赚取“猫粮”并累积“连胜”记录；答错则连胜清零、猫咪暴躁。将枯燥的记忆测试转化为有趣的互动游戏。

5. **绝对的隐私与极速体验**
   - **纯前端架构**：没有任何后端服务器，所有数据（包括图片）均通过 Base64 编码安全地保存在你的浏览器本地存储（Local Storage）中。
   - **数据自由**：内置完善的数据管理功能，支持一键导出备份（JSON）和导入恢复，你的数据完全由你掌控。

## 🛠 技术栈

- **核心框架**：React 18 (CDN 引入)
- **样式引擎**：Tailwind CSS (CDN 引入)
- **图标库**：Lucide React
- **部署方式**：单文件静态 HTML，零构建配置，开箱即用。
